第一步、React前端 与 服务器端建立链接

前言：

  • 简单理解socket.io中的on和 emit，相当于中央事件总线的on和emit；on表示接受对应事件，emit表示派发事件，两者是对应关系。
  • 服务端和客户端的链接是一对多的关系
  • 客户端 emit的事件只能被服务端的on接受，服务端的emit会被所有客户端的on接受
  • socket.io中的 connect与 disconnect事件是默认emit的
